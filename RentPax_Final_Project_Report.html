RentPax: AI-Powered Real Estate Investment Analysis Platform

Final Project Report

Joel Henry and Manan

Department of Computer Science

December 2024

Abstract—This paper presents RentPax, a comprehensive web-based real estate investment analysis platform designed to assist investors in making data-driven property investment decisions. The application integrates real-time property data from the RentCast API, advanced financial calculations including mortgage amortization, cap rate, and cash-on-cash return metrics, and artificial intelligence capabilities through Google's Gemini AI for investment recommendations. Built using modern web technologies including Next.js 14, TypeScript, PostgreSQL, and TailwindCSS, RentPax differentiates itself from existing solutions by offering a complete end-to-end solution that combines property valuation, comparable property analysis, affordability calculations based on debt-to-income ratios, 30-year financial projections, and personalized AI-driven investment advice. The platform was developed following an Agile software development methodology with iterative sprints, resulting in a production-ready application deployed on Vercel with a PostgreSQL database hosted on Railway. Testing demonstrates the system's ability to accurately calculate financial metrics and provide actionable investment insights to users.

Keywords—real estate investment, property analysis, financial modeling, artificial intelligence, web application, Next.js, TypeScript, machine learning, mortgage calculation, investment recommendation system

I. Introduction

Real estate investment remains one of the most popular wealth-building strategies in the United States, with the rental property market valued at over $4.5 trillion as of 2024 [1]. However, many potential investors face significant barriers to entry, including the complexity of financial analysis, difficulty in accessing reliable property data, and uncertainty about investment viability. Traditional methods of property analysis often require expensive software, professional consultants, or manual calculations that are prone to error [2].

This paper introduces RentPax, an innovative web-based platform that addresses these challenges by providing instant, comprehensive real estate investment analysis. The platform integrates multiple data sources and analytical capabilities into a single, user-friendly interface that enables both novice and experienced investors to evaluate rental properties with professional-grade accuracy.

The primary objectives of this project were to: (1) develop a scalable web application for property investment analysis, (2) implement accurate financial calculations following industry standards, (3) integrate artificial intelligence for personalized investment recommendations, (4) create an intuitive user interface accessible to users regardless of technical expertise, and (5) enable portfolio management for tracking multiple investment properties.

The remainder of this paper is organized as follows: Section II presents the literature survey and state of the art in real estate technology. Section III describes the software development process. Section IV details the system design. Section V covers the implementation. Section VI presents results and discussion. Section VII provides conclusions and future work.

II. Literature Survey and State of the Art

A. Real Estate Technology Landscape

The real estate technology (PropTech) industry has experienced significant growth, with global investment reaching $32 billion in 2023 [3]. This sector encompasses various applications including property search platforms, investment analysis tools, property management software, and automated valuation models (AVMs).

Automated Valuation Models represent a critical technology for property analysis. As described by Downie and Robson [4], AVMs use statistical modeling techniques to estimate property values based on comparable sales data, property characteristics, and market trends. Major providers like CoreLogic and Black Knight have developed sophisticated AVMs that are widely used by lenders and investors [5].

Zillow is the largest online real estate marketplace in the United States, offering property valuations through their proprietary Zestimate algorithm. While Zillow provides extensive property data, it lacks comprehensive investment analysis features such as cash flow calculations and return metrics [6].

Redfin offers similar property search and valuation capabilities with a focus on home buyers rather than investors. Their platform provides basic affordability tools but does not include detailed investment metrics [7].

BiggerPockets is a community-focused platform for real estate investors that provides educational content, forums, and basic calculators. While valuable for learning, their tools lack integration with real-time property data [8].

Mashvisor represents the closest competitor to RentPax, offering property analytics specifically for investors. However, Mashvisor requires expensive subscriptions starting at $199/month and does not include AI-powered recommendation features [9].

C. Where RentPax Fits

RentPax addresses a gap in the market by providing professional-grade investment analysis with the following differentiating features:

  1. AI-Powered Investment Recommendations: RentPax is the only platform in this comparison that integrates large language model AI (Google Gemini) to provide personalized investment recommendations with specific scores, risk assessments, and actionable suggestions [10].
  2. DTI-Based Affordability Analysis: Unlike competitors that provide basic affordability calculators, RentPax implements proper debt-to-income ratio calculations following lending industry standards (28% front-end, 36% back-end DTI ratios) as specified by federal mortgage guidelines [11].
  3. Comprehensive Free Tier: While competitors like Mashvisor charge premium prices, RentPax provides complete functionality without requiring paid subscriptions.
  4. Interactive Comparable Analysis: Users can select specific comparable properties to customize rent estimates, providing more accurate analysis than automated-only approaches.
  5. Multi-Fallback Architecture: The platform gracefully handles missing data through a multi-tier fallback system, ensuring functionality even without premium API access.

D. Artificial Intelligence in Real Estate

The application of artificial intelligence in real estate has grown significantly with advances in machine learning and natural language processing. Chen et al. [12] demonstrated that AI-based property valuation models can achieve accuracy comparable to human appraisers. More recently, large language models (LLMs) have been applied to provide contextual analysis and recommendations in financial domains [13].

RentPax leverages Google's Gemini AI model to analyze property metrics and generate investment recommendations. This approach goes beyond simple algorithmic scoring by providing natural language explanations of strengths, risks, and specific suggestions tailored to each property.

III. Software Development Process

A. Agile Methodology Overview

The development of RentPax followed an Agile software development methodology, specifically an adapted Scrum framework suitable for a two-person team. Agile was selected over traditional waterfall methodology due to its flexibility in accommodating changing requirements, emphasis on iterative development, and suitability for web application projects where user feedback is essential [14].

The Agile principles applied in this project included:

  • Iterative development with working software delivered at the end of each sprint
  • Regular sprint planning and retrospective meetings
  • Continuous integration and deployment
  • Adaptation to changing requirements and feedback
  • Face-to-face communication between team members

E. Security Design

Security was a primary consideration in the system design:

  • Password Security: All passwords are hashed using bcrypt with 10 salt rounds before storage
  • Session Management: JWT-based sessions with signed tokens using NEXTAUTH_SECRET
  • Input Validation: All API inputs validated using Zod schemas
  • SQL Injection Prevention: Prisma ORM with parameterized queries
  • HTTPS: Production deployment enforces TLS encryption

D. Key Algorithm Implementation

The core financial calculation algorithm implements standard mortgage amortization formulas. The monthly payment is calculated using:

M = P × [r(1+r)n] / [(1+r)n - 1]

Where M is the monthly payment, P is the principal loan amount, r is the monthly interest rate, and n is the total number of payments.

The implementation in TypeScript:

export function amortizedPI(loan: number, annualRate: number, years: number): number {
  const monthlyRate = annualRate / 100 / 12;
  const n = years * 12;
  if (monthlyRate === 0) return loan / n;
  return loan * (monthlyRate * Math.pow(1 + monthlyRate, n)) /
         (Math.pow(1 + monthlyRate, n) - 1);
}

D. Competitive Advantages Demonstrated

Through implementation and testing, RentPax demonstrated the following advantages over existing solutions:

  1. All-in-One Solution: Combines property valuation, financial analysis, AI recommendations, and portfolio management in a single platform, eliminating the need for multiple tools.
  2. AI-Powered Insights: The integration of Google Gemini AI provides human-like investment analysis that considers multiple factors and provides actionable recommendations—a feature not available in competing products.

  4. Free Access: Full functionality available without subscription fees, making professional-grade analysis accessible to all investors.
  5. Real Property Data: Integration with RentCast API provides actual market data rather than estimates, and interactive comparable selection allows user customization.
  6. Graceful Degradation: Multi-tier fallback system ensures application functionality even when external services are unavailable.

E. Limitations

Several limitations were identified during development and testing:

  • API Dependency: Accuracy depends on RentCast API data availability and quality
  • Geographic Coverage: Currently limited to US properties covered by RentCast
  • AI Rate Limits: Gemini API has rate limits that may affect heavy usage
  • Market Assumptions: 30-year projections assume constant appreciation and rent growth rates
